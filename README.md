# 数字电子技术 —— 课程复习总结

📖 基于课堂 PPT 整理的数字电子技术（数电）完整复习文档，涵盖 **第1~9章** 全部核心知识点、课堂例题、综合题目及考试技巧。

## 📋 内容概览

| 章节 | 主题 | 重点内容 |
|:---:|------|---------|
| 0 | 前言 | 🏠 用"小区门禁系统"类比理解数字电路 |
| 1 | 数字逻辑概论 | 数制转换、补码运算、8421BCD码、格雷码、基本逻辑运算 |
| 2 | 逻辑代数与化简 ⭐ | 基本定律、三大规则、公式法化简、**卡诺图化简法**（含无关项） |
| 3 | 逻辑门电路 | CMOS/TTL、OD门、三态门、传输门、扇出/噪声容限 |
| 4 | 组合逻辑电路 ⭐ | 分析方法、**设计方法**、编码器/译码器/MUX/加法器、竞争冒险 |
| 5 | 触发器与锁存器 | RS/JK/D/T 四种触发器特性方程、触发方式对比、Verilog描述 |
| 6 | 时序逻辑电路 ⭐ | **三大方程**、同步/异步分析、同步设计、74161/74194、任意进制计数器 |
| 7 | 半导体存储器 | ROM/RAM分类、SRAM vs DRAM、容量扩展 |
| 8 | 脉冲波形的产生 | 555定时器三种电路（施密特/单稳态/多谐振荡器）及核心公式 |
| 9 | 数模与模数转换 | DAC输出电压计算、ADC四步骤、三种ADC对比、采样定理 |
