0. 前言——用类比理解数字电路

🏠 用"小区门禁系统"类比数字电路

想象你要设计一个小区门禁系统：

门禁系统的需求对应的数字电路概念
住户刷卡开门，卡号是一串数字数制与编码 —— 用0/1表示信息
"同时刷两张卡+输入密码"才能进与逻辑 —— 条件全部满足才开门
"刷卡 或 输密码"都能开门或逻辑 —— 任一条件满足即可
黑名单人员刷卡不开门非逻辑 —— 条件取反
记录最近3次进出记录触发器/寄存器 —— 存储状态
统计进入人数，满100人禁止进入计数器 —— 累计计数
把复杂的开门条件化简为最少硬件逻辑化简 —— 卡诺图/公式法

某通信系统每秒传输 1.544M 位数据，求每位数据的时间。

2.1 逻辑代数基础

基本定律和恒等式

类别公式名称
变量与常量A+0=A, A·1=A自等律
A+1=1, A·0=00-1律
A+A̅=1互补律
A·A̅=0
与普通代数相似A+B=B+A, A·B=B·A交换律
A+(B+C)=(A+B)+C结合律
A(B+C)=AB+AC分配律
特殊定律A+A=A, A·A=A重叠律（无指数）
A+AB=A吸收律
A+A̅B=A+B消因子律
AB+A̅C+BC=AB+A̅C冗余项定理
摩根定理(A·B)̅ = A̅ + B̅非常重要！
(A+B)̅ = A̅ · B̅

🔑 三个基本规则

  1. 代入规则：任何含有变量A的等式中，将所有A代之以同一个逻辑函数，等式仍成立。
  2. 反演规则：求一个逻辑函数L的反函数 L̅ 时：
    • 所有"·"变"+"，"+"变"·"
    • "0"变"1"，"1"变"0"
    • 原变量变反变量，反变量变原变量
    • 注意运算优先级！
  3. 对偶规则：求对偶式L'时（不改变变量）：
    • "·"变"+"，"+"变"·"；"0"变"1"，"1"变"0"
    • 若两逻辑式相等，则其对偶式也相等

📝 反演规则举例

L = A·B + C̅·D，求 L̅。

L̅ = (A̅ + B̅) · (C + D̅)

注意：原来 AB 和 C̅D 之间是"+"，取反后变成"·"，需要加括号保持运算优先级！

2.2 逻辑函数的代数化简法

常用化简方法

  1. 并项法：利用 AB+AB̅=A，将两项合并为一项
  2. 吸收法：利用 A+AB=A，消去多余项
  3. 消去法：利用 A+A̅B=A+B，消去多余因子
  4. 配项法：利用 A=A(B+B̅)，先展开再化简

📝 公式法化简示例

化简：L = AB + A̅C + BC

解：L = AB + A̅C + BC(A+A̅) = AB + A̅C + ABC + A̅BC

= (AB + ABC) + (A̅C + A̅BC)

= AB(1+C) + A̅C(1+B)

= AB + A̅C（这就是冗余项定理：第三项BC是冗余的）

📝 公式法化简示例2

化简：L = ABC + ABC̅ + A̅B

解：前两项合并：ABC + ABC̅ = AB(C+C̅) = AB

L = AB + A̅B = (A+A̅)B = B

💡 公式法化简经验总结（来自课堂PPT）：
  1. 先用吸收法 A+AB=A 去掉多余的与项
  2. 再用并项法 AB+AB̅=A 合并互补项
  3. 用消去法 A+A̅B=A+B 消去多余的因子
  4. 必要时配项法 A=A(B+B̅) 展开再化简
  5. 卡诺图法更直观！——当公式法做到一半做不下去时，果断换卡诺图

2.3 卡诺图化简法 考试必考！

🔑 最小项与卡诺图

  • 最小项：n个变量的逻辑函数中，包含所有变量的乘积项（与项），每个变量以原变量或反变量形式出现一次且仅一次。
  • 编号：原变量=1，反变量=0，转为十进制编号 mi。如 ABC(111) → m₇
  • 最小项之和：任何逻辑函数都可写成最小项之和的标准形式

卡诺图的构成（相邻性——循环邻接）

变量数卡诺图结构
2变量2×2 格，行A，列B
3变量2×4 格，行A，列BC（格雷码序：00 01 11 10）
4变量4×4 格，行AB，列CD（均为格雷码序）

关键：卡诺图的行列排列使用格雷码顺序（00→01→11→10），确保相邻格只有一位变量变化。注意上下边界和左右边界也相邻（循环邻接）。

🔑 卡诺图化简步骤

  1. 将逻辑函数填入卡诺图（每个最小项对应格填1，其余填0）
  2. 画卡诺圈：圈住所有的1，圈要尽量大（每个圈含2ⁿ个相邻格），允许重叠
  3. 每个圈写出一个乘积项（消去变化的变量，保留不变的变量）
  4. 所有圈对应的乘积项相加，得最简与或式

📝 卡诺图化简示例

用卡诺图化简：L(A,B,C,D) = Σm(0,2,4,6,8,10,12,14)

分析：观察这些最小项，CD列中 C=0,D=0 和 C=1,D=0 行（即D=0）都有1。

       CD
AB   00  01  11  10
00  | 1 | 0 | 0 | 1 |   ← m₀, m₂
01  | 1 | 0 | 0 | 1 |   ← m₄, m₆
11  | 1 | 0 | 0 | 1 |   ← m₁₂, m₁₄
10  | 1 | 0 | 0 | 1 |   ← m₈, m₁₀

圈起所有为D̅的格（即CD=00和CD=10这两列）：结果是 L = D̅

经验：如果化简结果简单到只有一个变量，说明原函数本身就很简单。

2.4 具有无关项（约束项）的化简

  • 约束项/无关项：输入变量的某些取值组合永远不会出现（或出现后输出无所谓），用"×"或"d"表示
  • 化简时：无关项既可以当作1来扩大卡诺圈，也可以当作0忽略
  • 目标：利用无关项使化简结果最简

⚠️ 化简注意事项

  • 卡诺圈必须覆盖所有1，无关项可选
  • 圈尽可能大（2ⁿ格），数量尽可能少
  • 不要漏掉边界相邻的情况
  • 化简结果不唯一时，选最简的

第3章 逻辑门电路

🎯 本章重点：MOS管开关特性、CMOS反相器/与非门/或非门结构、TTL门电路特性、OD门与三态门、门电路使用中的实际问题（扇出、噪声容限、传输延迟）。

3.1 MOS逻辑门电路

MOS管作为开关

  • NMOS：VGS > VTN（阈值电压）→ 导通（开关闭合）；否则截止（开关断开）
  • PMOS：VGS < VTP（负阈值）→ 导通；否则截止
  • CMOS：NMOS和PMOS互补配对，一个导通时另一个截止 → 静态功耗极低

🔑 CMOS基本门电路

门类型结构特点
CMOS反相器PMOS在上，NMOS在下，栅极相连为输入最基本的CMOS门
CMOS与非门两个PMOS并联在上，两个NMOS串联在下串联NMOS增加→多个输入的与非
CMOS或非门两个PMOS串联在上，两个NMOS并联在下串联PMOS增加→多个输入的或非

3.2 TTL逻辑门电路（了解）

  • TTL（Transistor-Transistor Logic）：双极型晶体管构成
  • 特点：速度较快，但功耗大于CMOS
  • 输入级：多发射极晶体管
  • 输出级：推挽式（图腾柱）输出

3.3 CMOS门电路的不同输出结构

🔑 三种特殊输出结构

类型特点应用
OD门（漏极开路）输出端需外接上拉电阻线与连接、电平转换
三态门（TS门）有使能端EN，输出有三种状态：0、1、高阻态(Z)总线连接、双向I/O
传输门（TG）双向传输，由NMOS和PMOS并联构成模拟开关、多路选择器

3.4 逻辑门电路使用中的实际问题

主要参数

  • 扇出系数 NO：一个门能驱动同类门的最大数目
  • 噪声容限：输入端允许的噪声干扰电压范围
    • 低电平噪声容限 VNL = VIL(max) − VOL(max)
    • 高电平噪声容限 VNH = VOH(min) − VIH(min)
  • 传输延迟时间 tpd：信号从输入到输出的平均延迟
  • 功耗：静态功耗 + 动态功耗（CMOS动态功耗 ∝ f）

第5章 触发器和锁存器

🎯 本章重点：锁存器与触发器的区别、RS/JK/D/T四种触发器的特性方程和功能表、电平触发/边沿触发/主从触发的区别、异步清零和置位、Verilog描述。
💡 触发器 = 有记忆功能的基本单元
触发器的核心：能存储1位二进制信息（0或1）。
锁存器 vs 触发器：锁存器没有时钟控制（电平敏感），触发器有时钟控制（边沿触发）。
电平触发 vs 边沿触发：电平触发在整个时钟有效电平期间都可能改变状态；边沿触发只在时钟跳变瞬间采样，抗干扰能力强。

5.1 基本RS锁存器

与非门构成的RS锁存器

S̅R̅Q状态
11保持保持原状态
011置1（Set）
100置0（Reset）
00不定禁止！

约束条件：S̅ + R̅ = 1（即 S̅和R̅ 不能同时为0）

特性方程：Qn+1 = S + R̅Qn（注意是S不是S̅）

📝 例 消除机械开关抖动

机械开关在切换时会产生抖动，用基本RS锁存器可以消除。

原理：开关打到上触点→S̅=0→Q立即置1且保持，后续的抖动（S̅在0/1之间跳）不会改变Q的状态。

5.2 时钟触发的触发器

触发方式对比

触发方式特点抗干扰
电平触发CP=1期间输出随输入变化差（存在"空翻"问题）
主从触发CP=1主触发器变化，CP↓从触发器跟随较好（但有时仍可能"一次变化"）
边沿触发仅在CP↑（或CP↓）瞬间采样输入最好（推荐使用）

5.3 四种触发器的特性方程 必背！

🔑 触发器特性方程汇总

类型特性方程功能表特殊功能
RS触发器 Qn+1 = S + R̅Qn
约束：RS=0		 S=0,R=0→保持；S=1,R=0→置1
S=0,R=1→置0；S=1,R=1→禁止		 有约束条件
JK触发器 Qn+1 = JQ̅n + K̅Qn J=0,K=0→保持；J=0,K=1→置0
J=1,K=0→置1；J=1,K=1→翻转		 无约束！J=K=1时翻转
D触发器 Qn+1 = D D=0→置0；D=1→置1 最简单，输出=输入（延迟一拍）
T触发器 Qn+1 = T⊕Qn T=0→保持；T=1→翻转 用于计数器

💡 触发器之间的相互转换

  • JK → D：J=D, K=D̅（代入特性方程：JQ̅+K̅Q = DQ̅+DQ = D）✓
  • JK → T：J=K=T（代入：TQ̅+T̅Q = T⊕Q）✓
  • D → T：D = T⊕Qn
  • D → JK：D = JQ̅n + K̅Qn

📝 课堂练习

试分别用JK和D触发器实现特性方程为 Qn+1 = X⊕Qn 的触发器（T触发器）。

解：

  • 用JK触发器：令 J=K=X（即T触发器是JK触发器的特例，J=K=T）
  • 用D触发器：D = X⊕Qn（加一个异或门）

5.4 有异步清零/置位端的集成触发器

  • 异步清零 Rd（或CR）：Rd=0时，Q立即被清零（与CP无关）
  • 异步置位 Sd（或PR）：Sd=0时，Q立即被置1（与CP无关）
  • 正常工作时 Rd = Sd = 1
  • 集成JK触发器常有多输入端：J=J₁·J₂，K=K₁·K₂

5.5 Verilog HDL描述

关键语法

  • 电平敏感（锁存器）：always @(E or D)
  • 边沿敏感（触发器）：always @(posedge CP or negedge CR)
  • 非阻塞赋值 <=：并行执行，用于时序逻辑
  • 阻塞赋值 =：顺序执行，用于组合逻辑

📝 D触发器的Verilog代码

module DFF (Q, D, CP);
  output Q;  input D, CP;  reg Q;
  always @(posedge CP)
    Q <= D;          // CP上升沿，Q取D的值
endmodule

6.1 时序逻辑电路的基本概念

🔑 时序电路的三大方程

方程类型表达式含义
输出方程O = f₁(I, S)输出与输入、现态的关系
激励方程（驱动方程）E = f₂(I, S)各触发器输入端的逻辑表达式
状态转换方程（次态方程）Sn+1 = f₃(E, Sn)将激励代入特性方程得到次态

⚠️ 三大方程——是时序电路分析和设计的核心！

具体做法（以JK触发器为例）：

  1. 激励方程：从电路图中写出各触发器J、K端的逻辑表达式。如 J₁=X⊕Q₀, K₁=1
  2. 状态转换方程：将激励方程代入触发器的特性方程。如 Q₁ⁿ⁺¹ = J₁Q̅₁ⁿ + K̅₁Q₁ⁿ = (X⊕Q₀)Q̅₁ⁿ + 0·Q₁ⁿ
  3. 输出方程：从电路图中写出输出与现态/输入的关系。如 Y=Q₁Q₀

考试必考！无论是分析题还是设计题，三大方程是第一步，写错了后面全错。

时序电路的分类

分类方式类型特点
按时钟同步时序电路所有触发器使用同一CP（重点掌握）
异步时序电路各触发器CP不同
按输出米利型 Mealy输出 = f(输入, 状态)

6.2 同步时序逻辑电路的分析

🔑 分析步骤（五步法）

  1. 了解电路组成：确定输入、输出、触发器类型和个数
  2. 写出三大方程：输出方程、激励方程、状态转换方程（将激励代入特性方程）
  3. 列出状态转换表：逐一代入现态和输入，求出次态和输出
  4. 画出状态转换图 / 时序图
  5. 分析逻辑功能：根据图表总结电路功能

📝 例1 分析T触发器构成的同步时序电路

电路由两个T触发器组成，激励方程为：T₀=A，T₁=AQ₀，输出Y=AQ₁Q₀。

状态转换方程（代入T触发器特性 Qn+1=T⊕Qn）：

Q₀n+1 = A ⊕ Q₀n
Q₁n+1 = (AQ₀) ⊕ Q₁n

功能：当A=1时进行加1计数（00→01→10→11→00），当A=0时停止计数。是一个A控制的二进制计数器。

📝 例2 脉冲分配器

分析三个D触发器构成的同步时序电路，输出Z₀=Q₀, Z₁=Q₁, Z₂=Q₂。

功能：有效状态为三位循环码（000→001→011→111→110→100→000），各触发器的Q端轮流输出一个脉冲，循环周期为6TCP。这是一个脉冲分配器（节拍脉冲产生器）。

📝 例3 可逆计数器（JK触发器实现）

由两个JK触发器组成，J₂=K₂=X⊕Q₁, J₁=K₁=1, 输出Y=Q₂Q₁。

功能：

  • X=0时：电路进行加1计数（00→01→10→11→00）
  • X=1时：电路进行减1计数（00→11→10→01→00）
这是一个可逆二进制计数器，Y可作为进位/借位信号。

6.3 同步时序逻辑电路的设计 重点难点！

🔑 设计步骤

  1. 逻辑抽象：根据设计要求确定输入/输出变量、电路状态数及其含义，画出原始状态转换图
  2. 状态化简：消去等价状态（输入相同→输出相同→次态相同），得最简状态图
  3. 状态编码：给每个状态分配二进制代码（编码方案影响电路复杂度）
  4. 选择触发器类型：确定个数 n（2ⁿ⁻¹ < 状态数 ≤ 2ⁿ）和类型（D/JK常用）
  5. 求激励方程和输出方程：利用卡诺图化简
  6. 画逻辑图
  7. 检查自启动能力：检查未用状态能否进入有效循环（不能则需要修改）

📝 例1 用D触发器设计8421BCD码同步十进制加计数器

设计要点：

  • 状态：0000→0001→...→1001→0000（共10个有效状态）
  • 用4个D触发器（因为2³=8<10≤16=2⁴）
  • 根据状态转换表写次态方程，D触发器特性为 Qn+1=D，所以 D=Qn+1
  • 用卡诺图化简求出 D₃, D₂, D₁, D₀ 的最简表达式
  • 检查自启动：无效状态 1010~1111 是否能进入有效循环

📝 例2 序列检测器（检测110）

输入X为串行数据，当检测到X输入"110"序列时输出Z=1。

设计过程：

  1. 定义状态：a=初始(收到0个正确位)，b=收到"1"，c=收到"11"，d=收到"110"(输出1)
  2. 状态化简：a和d在相同输入下输出和次态相同→合并。最终3个状态：a, b, c
  3. 状态编码：a=00, b=01, c=11（2个JK触发器）
  4. 列状态转换表→卡诺图化简→得激励方程和输出方程
  5. 检查：状态10未用，分析其是否自启动

📝 例3 自动售货机控制器

投币口每次可投入一分或二分硬币，投入三分给出一盒火柴，投入四分给出火柴的同时找回一分。

设计思路：

  • 输入：A=1（二分硬币），B=1（一分硬币）
  • 输出：Y=1（给火柴），Z=1（找一分）
  • 状态：S₀=未投币，S₁=已投一分，S₂=已投二分
  • 用2个D触发器，状态编码：S₀=00, S₁=01, S₂=10

🔑 任意进制计数器的设计方法

清零法（复位法）：

  1. 写出模N计数器的状态SN的二进制编码
  2. 将等于1的Q端通过与非门接到CR端（异步清零型）或同步清零端
  3. 异步清零：N状态短暂出现后立即清零（可能出现毛刺）
  4. 同步清零：在N-1状态时准备好清零信号

置数法：

  1. 选择合适的预置数起始值，使计数范围恰好为N个状态
  2. 在终止状态通过逻辑门产生置数控制信号（PE/LD）
  3. 74LS161为同步置数：在CP上升沿时将预置数装入

📝 用74LS161设计模6计数器（清零法）

74LS161是异步清零（CR=0立即清零）。

模6需要状态：0→1→2→3→4→5→(6→0)。

当 Q₃Q₂Q₁Q₀ = 0110（即状态6）时，Q₂=1, Q₁=1 → 通过与非门使 CR=0 → 立即清零回0000。

📝 用74LS161设计模6计数器（置数法）

利用同步置数功能，计数范围设为 0→1→2→3→4→5→(置数→0)。

当 Q₃Q₂Q₁Q₀ = 0101（状态5）时 → PE=0（置数使能有效）→ 下一个CP来时装入D₃D₂D₁D₀=0000。

或用另一种方案：置入10(1010)，计数 10→11→12→13→14→15→(置数→10)，恰好6个状态。

📝 课堂例题：用74161构成十进制加计数器（来自PPT）

复位法：74161异步复位，当状态为1010（十进制的10）时→与非门产生CR=0→立即清零回0000。有效状态0~9共10个。

置数法：74161同步置数，利用LD端，当计数到1001(9)时→LD=0→下一CP装入0000。或置入初值0110(6)，计数6→7→...→15后进位TC使LD有效→置回6，恰好10个状态。

📝 课堂例题：用74161构成24进制加计数器（来自PPT）

24 = (11000)B，有效循环 00000~10111。两片74161级联，当计数到11000(24)时→与非门→两片CR同时清零。

💡 计数器扩展方法（来自PPT）

  • 同步扩展（级联）：低位片的进位TC接高位片的CEP/CET使能端，两片共用同一CP。当低位计满(1111)时TC=1，下一个CP高位片才加1。
  • 异步扩展：低位片的进位输出作为高位片的CP时钟。简单但有累积延迟。
  • N个模M计数器级联 = M的N次方进制。如2片74LS160(模10)级联=100进制。

7490 异步十进制计数器（补充）

  • CP1为脉冲，Q3Q2Q1输出 → 五进制计数器（000→001→010→011→100）
  • 将Q0接CP1 → 十进制计数器（8421BCD码）

      2'b01: Q <= {Q[2:0], Dsr};  // 右移
      2'b10: Q <= {Dsl, Q[3:1]};  // 左移
      2'b11: Q <= D;              // 并行输入
    endcase
endmodule

module counter74x161 (CEP, CET, PE, CP, CR, D, TC, Q);
  input CEP, CET, PE, CP, CR;

📝 序列检测器（状态图）的Verilog建模

module Mealy_sequence_detector (A, CP, CR, Y);
  input A, CP, CR;
  output reg Y;
  reg [1:0] current_state, next_state;
  parameter S0=2'b00, S1=2'b01, S2=2'b11;
  // 状态更新
  always @(negedge CP or negedge CR)
    if (~CR) current_state <= S0;
    else     current_state <= next_state;
  // 次态逻辑和输出
  always @(current_state or A) begin
    case(current_state)
      S0: begin Y=0; next_state = (A==1)? S1 : S0; end
      S1: begin Y=0; next_state = (A==1)? S2 : S0; end
      S2: if (A==1) begin Y=0; next_state = S2; end
          else      begin Y=1; next_state = S0; end
      default: begin Y=0; next_state = S0; end
    endcase
  end
endmodule

上述代码描述了一个米利型110序列检测器。

🔑 Verilog时序逻辑编程要点总结

  • 状态更新用 always @(posedge CP or negedge CR) + 非阻塞赋值 <=
  • 次态逻辑可单独写组合逻辑 always @(*)
  • FSM三段式写法：状态更新（时序）+ 次态逻辑（组合）+ 输出逻辑（组合/时序）
  • 参数定义状态：用 parameter 定义状态编码，推荐格雷码或独热码

8.4 555定时器 重点器件！

🔑 555定时器内部结构

2个电压比较器（参考电压分别为 2/3 VCC 和 1/3 VCC）+ 基本RS触发器 + 放电管（T）+ 3个5kΩ分压电阻

关键引脚：

  • ⑥ TH（阈值）：接比较器1，阈值 2/3 VCC
  • ② TR（触发）：接比较器2，阈值 1/3 VCC
  • ⑦ DIS（放电）：放电管集电极，用于电容充放电
  • ④ RD（复位）：低电平有效，强制输出为0
  • ⑤ CO（控制电压）：可外接电压改变阈值

🔑 555构成多谐振荡器——核心公式（必背！）

参数公式
充电时间（高电平脉宽）tw1 = 0.693 (R₁+R₂)C ≈ 0.7 (R₁+R₂)C
放电时间（低电平脉宽）tw2 = 0.693 R₂C ≈ 0.7 R₂C
周期 TT = tw1 + tw2 = 0.693 (R₁+2R₂)C
频率 ff = 1/T = 1.443 / [(R₁+2R₂)C]
占空比 qq = (R₁+R₂) / (R₁+2R₂) ⚠️ 恒大于50%

记忆技巧：充电经R₁+R₂，放电只经R₂（因T导通旁路了R₁）。所以充电时间>放电时间，占空比始终>50%。

📝 例 多谐振荡器参数计算

R₁=20kΩ，R₂=10kΩ，C=0.1μF，求 T、f、q。

解：

  • T = 0.693 × (20+20)×10³ × 0.1×10⁻⁶ = 0.693 × 40×10³ × 10⁻⁷ = 2.77 ms
  • f = 1 / 2.77ms ≈ 361 Hz
  • tw1 = 0.693 × (20+10)×10³ × 10⁻⁷ = 2.08 ms（高电平）
  • q = (20+10)/(20+20) = 30/40 = 75%

💡 如何使占空比 ≤ 50%？

在R₂两端并联一个二极管（方向：正极接DIS端）。充电经R₁+二极管，放电经R₂，使充电路径电阻 < 放电路径电阻。

解：

关键原理：74LS138输出 Y̅ᵢ = m̅ᵢ（对应最小项的非）。

F = m₁+m₂+m₄+m₇ = (Y̅₁·Y̅₂·Y̅₄·Y̅₇)̅

将A、B、C接地址端A₂、A₁、A₀，Y̅₁、Y̅₂、Y̅₄、Y̅₇ 接与非门74LS20输入，输出即得F。

📝 综合题11：用8选1数据选择器74LS151实现逻辑函数

用74LS151实现 F(A,B,C) = Σm(2,3,5,6)。

点击查看解答

ABC接地址端A₂A₁A₀。数据端 D₂=D₃=D₅=D₆=1，其余 Dᵢ=0。E̅=0选通。

原理：MUX本质是一个"电子开关"，根据地址选择对应的数据输入端输出。

📝 综合题12：竞争冒险——判断与消除

判断 F = AB + A̅C 在什么条件下可能存在竞争冒险？如何消除？

点击查看解答

当 B=C=1 时：F = A + A̅ → 若A和A̅到达时间不同，出现短暂的0脉冲。

消除：增加冗余项 BC → F = AB + A̅C + BC。当B=C=1时BC=1保持输出稳定。

📋 各章重点汇总

第1章 数字逻辑概论

  • ✅ 数制转换：二↔十↔十六进制（必考）
  • ✅ 补码运算：正数补码=原码；负数补码=反码+1（减法转加法）
  • ✅ 8421BCD码：4位二进制表示1位十进制，9以上的1010~1111不使用
  • ✅ 格雷码：相邻码仅差一位；与二进制互转公式
  • ✅ 三种基本逻辑：与（全1出1）、或（有1出1）、非（取反）
  • ✅ 逻辑函数五种表示方法：真值表、表达式、逻辑图、波形图、卡诺图

第2章 逻辑代数与化简

  • ✅ 基本定律：摩根定理 (A·B)̅ = A̅+B̅，(A+B)̅ = A̅·B̅
  • ✅ 三个规则：代入、反演、对偶
  • ✅ 最小项：n个变量的乘积项，每个变量出现一次（编号mᵢ）
  • ✅ 卡诺图化简（重点）：格雷码顺序排列→画最大的圈→消去变化变量→写出最简与或式
  • ✅ 含无关项的化简：无关项×可当1用，帮助扩大圈
  • ✅ 竞争与冒险：卡诺圈相切→增加冗余项消除

第3章 逻辑门电路

  • ✅ CMOS：低功耗、NMOS与PMOS互补
  • ✅ OD门：漏极开路，需外接上拉电阻，可实现线与
  • ✅ 三态门：0/1/高阻(Z) 三态，用于总线
  • ✅ 传输门：双向传输，模拟开关
  • ✅ 扇出、噪声容限、传输延迟等实际参数

第4章 组合逻辑电路

  • ✅ 分析步骤：逻辑图→表达式→真值表→功能
  • ✅ 设计步骤：要求→真值表→表达式→化简→逻辑图
  • ✅ 编码器(74LS148)、译码器(74LS138)：互为逆过程
  • ✅ 数据选择器 MUX："多选一"，可实现任意逻辑函数
  • ✅ 加法器：半加器（本位和+进位）、全加器（考虑低位进位）
  • ✅ 用MSI（中规模）器件实现组合逻辑

第5章 触发器与锁存器

  • ✅ 四种触发器特性方程（必背！）：
    • RS：Qn+1 = S + R̅Qn，约束 RS=0
    • JK：Qn+1 = JQ̅n + K̅Qn
    • D：Qn+1 = D
    • T：Qn+1 = T⊕Qn
  • ✅ 边沿触发 > 主从触发 > 电平触发（抗干扰能力）
  • ✅ 异步清零/置位：与CP无关，立即生效
  • ✅ 触发器间的相互转换

第6章 时序逻辑电路（重中之重）

  • ✅ 三大方程：输出方程、激励方程、状态转换方程
  • ✅ 同步时序分析：写方程→列表→画图→析功能
  • ✅ 同步时序设计：状态图→化简→编码→选触发器→求方程→画图→自启动检查
  • ✅ 74LS161计数器：异步清零+同步置数，任意进制设计（清零法/置数法）
  • ✅ 74LS194移位寄存器：S₁S₀控制：保持(00)/右移(01)/左移(10)/置数(11)
  • ✅ 任意N进制计数器 = M进制计数器 + 清零/置数逻辑
  • ✅ 序列发生/检测器设计

第7章 半导体存储器

  • ✅ ROM：掩膜/PROM/EPROM/EEPROM/Flash
  • ✅ SRAM vs DRAM：静态(触发器) vs 动态(电容)，需刷新 vs 不需
  • ✅ 容量扩展：位扩展(并联数据线)、字扩展(增加地址线+片选)
  • ✅ 总片数 = (目标字/单片字) × (目标位/单片位)

第8章 脉冲波形的变换与产生

  • ✅ 施密特触发器：双阈值 + 滞回特性（回差 ΔVT=VT+−VT-）
  • ✅ 单稳态触发器：一触即发，固定脉宽 tw=1.1RC
  • ✅ 多谐振荡器：自激振荡，无需触发，产生方波
  • ✅ 555定时器核心公式（必背）：
    • 多谐振荡器：T=0.693(R₁+2R₂)C，q=(R₁+R₂)/(R₁+2R₂) > 50%
    • 单稳态：tw = 1.1 RC
    • 施密特：VT+=2/3VCC，VT-=1/3VCC
  • ✅ 三种电路识别：看TH(⑥)和TR(②)的接法即可区分

第9章 数模与模数转换器

  • ✅ DAC公式：VO = −(VREF/2n) × D
  • ✅ DAC分辨率：1/(2n−1)，n越大精度越高
  • ✅ ADC四步骤：采样→保持→量化→编码
  • ✅ 采样定理：fs ≥ 2fi(max)
  • ✅ 三种ADC：并行比较型（最快）、逐次逼近型（最常用）、双积分型（最准）

💡 考试与复习技巧

  1. 卡诺图化简是每年必考，4变量卡诺图5分钟以内完成
  2. 时序电路分析按五步法逐项写，不要跳步，即使结果错误，步骤分也能拿到
  3. 时序电路设计最容易被卡住的地方是状态分配，优先用格雷码减少竞险
  4. 任意进制计数器掌握清零法和置数法两种套路，考场上二选一
  5. 触发器特性方程记牢，特别是JK和D——所有的状态转换都从它们展开
  6. 74LS161/74LS194的功能表记住，MSI设计题全靠它们
  7. Verilog题注意阻塞赋值(=)和非阻塞赋值(<=)的区别，always敏感量别漏
  8. 检查自启动——设计题的最后一步，忘了会扣分

⚠️ 常见错误提醒

  1. 摩根定理变换后忘了加括号
  2. 卡诺图行列没按格雷码排列
  3. 异步清零/置位忘记标"↓"表示低电平有效
  4. 时序电路分析时把激励方程和状态方程混为一谈
  5. 任意进制清零法设计时，忘记异步清零是N状态触发而非N-1
  6. Verilog里敏感量表只写了posedge CP而忘了negedge CR
